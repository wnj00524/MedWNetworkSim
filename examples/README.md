# Examples

These JSON files are standalone sample networks for MedWNetworkSim.

Load them through the app's `Open` action.

`New` creates a blank network and does not load any sample content.

Files in this folder:

- `minimal-small-example.json`: a tiny two-node network for quick editing and connectivity checks
- `avalonia-migration-pilot.json`: the former Avalonia seeded sample, now provided as an explicit file instead of being injected by `New`
